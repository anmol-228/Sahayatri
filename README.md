# Shahyatri – Smart Transport Web Application (Prototype)

## Overview
Shahyatri is a frontend-based prototype developed during Smart India Hackathon to improve urban public transport usability through structured route discovery and navigation visualization.

The system consists of two modules:

- user-app – Commuter interface  
- driver-app – Driver interface  

## Problem Statement
Urban commuters often face difficulty in identifying routes, understanding transport flow, and accessing structured transport information.

## Features
- Route interface design  
- Map-based visualization using OpenStreetMap  
- Separate user and driver modules  
- Responsive UI using Bootstrap  

## Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap  
Backend (Planned): Spring Framework  
Database (Planned): MySQL  
Maps Integration: OpenStreetMap  

## Current Status
This project is a working prototype demonstrating UI flow and system architecture planning. Backend integration and real-time data handling are part of future scope.
