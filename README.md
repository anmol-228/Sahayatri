# Shahyatri – Smart Transport Web Application (Prototype)

## Overview
Shahyatri is a frontend-based prototype developed during Smart India Hackathon.

The system consists of two modules:
- user-app (Commuter Interface)
- driver-app (Driver Interface)

## Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Maps Integration: OpenStreetMap

## Current Status
This is a prototype demonstrating UI flow and architecture planning.
